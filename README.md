# CS455 Agent

The Agent is a Python script for collecting data from the hardware.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install external module.

```bash
pip install --update
pip install psutil
pip install json
pip install uuid
pip install logging
```
