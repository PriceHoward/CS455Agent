What is in the json File?
The first line is the CPU usage over 15 minutes of being used.
The second line is the Disk usage split as: Total Disk, Used Disk, Free Disk, Percentage used.
The third line is Memory Usage slit as: Total Memory, Availabale Memory, Percent Used, Used, Free, Active, Inactive, Wired.
The fourth line is Network Usage split as: Bytes_Sent, Bytes_Recieved,Packets_Sent, Packets_Recieved, Errin, Errout, Dropin, Dropout
